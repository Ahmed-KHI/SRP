Usage Patterns
1. Command Line Tool

Most common usage for daily receipt processing

# Process single receipt
python main.py process receipt.jpg

# Process folder
python main.py process-folder receipts/

# Export to Excel
python main.py export expenses.xlsx
2. Python Library

Integration with existing applications

# Import framework
from core.receipt_processor import ReceiptProcessor

# Initialize
processor = ReceiptProcessor(config)

# Process receipts
result = processor.process_receipt('receipt.jpg')
receipts = processor.process_folder('receipts/')

# Export results
processor.export_to_excel(receipts, 'report.xlsx')
